create DGZ of arbitrary shapes from C:\usr\local\lib\curves\testclip.tcl


steps for printing object

1. choose shape in SVGCreator
2. hit save twice (to make two copies). make sure it created a .svg file in this directory and saved two new rows to l:\stimuli\grasp\objects.db table "objectsTable"
3. open template in Fusion 360 "addSVGtoStem in Grasparatus > Print Queue
4. insert svg, extrude 5 mm, save
5. get all your objects into one file that you want to print, export as stp
6. in freecad, convert to stl
7. send to 3d printer, print front face down
