thingSpeakAuthenticate

Store read/write API key

Syntax

thingSpeakAuthenticate(username,password)

thingSpeakAuthenticate(chId,Name,Value)

Description

thingSpeakAuthenticate(username,password) authenticates all the channels in the specified user account. username can be either a MathWorks® account login ID, a ThingSpeak™ account login ID, or an email ID. After authentication, you do not have to provide the read key or write key for any of the channels contained in this user account.

thingSpeakAuthenticate(chId,Name,Value) allow authentication to only the channel specified by chId with optional Name,Value pair arguments.

Input Arguments

username - username can be either be your MathWorks account login, ThingSpeak login, or email ID, specified as a character vector.

password - Your password for the associated account, specified as a character vector.

chID - The channel identification number, specified as a positive integer.

Examples

Authenticate using channel ID and API keys.

thingSpeakAuthenticate(17504, ...
    'ReadKey','TJW6Y321SUGLKPGR','WriteKey','23ZLGORQN9TWHG2H');

Authenticate with username and password.

userName = 'Enter ThingSpeak.com user name';
passwd   = 'Enter ThingSpeak.com password';
thingSpeakAuthenticate(userName,passwd);

Authenticate with email ID and password.

emailID = 'Enter ThingSpeak.com email ID';
passwd  = 'Enter ThingSpeak.com password';
thingSpeakRead(emailID,passwd);

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN.

Name

Description

Data type

'ReadKey'

Read API Key of the channel.

character vector

'WriteKey'

Write API Key of the channel.

character vector

Both the read and write API keys can be found on ThingSpeak.com.
