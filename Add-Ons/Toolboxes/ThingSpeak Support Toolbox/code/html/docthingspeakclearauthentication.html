thingSpeakClearAuthentication

Clear authentication information

Contents

  • Syntax
  • Description
  • See Also

Syntax

thingSpeakClearAuthentication

thingSpeakClearAuthentication(chId)

thingSpeakClearAuthentication(username)

thingSpeakClearAuthentication(email)

Description

thingSpeakClearAuthentication clears all authentication information stored for the current session of MATLAB®. The cleared information includes read/write API keys and username/email ID based authentication information.

thingSpeakClearAuthentication(chId) clears both read and write API keys stored for the specified chId.

thingSpeakClearAuthentication(username) clears authentication to all channels owned by the specified username account. Note that thingSpeakRead can still access the public channels owned by the specified username account.

thingSpeakClearAuthentication(email) clears authentication to all channels owned by the specified email account. Note that thingSpeakRead can still access the public channels owned by the specified email account.
