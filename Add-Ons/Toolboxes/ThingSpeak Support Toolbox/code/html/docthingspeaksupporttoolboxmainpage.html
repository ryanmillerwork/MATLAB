ThingSpeak Support Toolbox

ThingSpeak™ is a free web service hosted by MathWorks® that lets you collect, analyze and act on sensor data, and develop Internet of Things applications. The ThingSpeak Support Toolbox lets you use MATLAB® to read data from ThingSpeak and write data to ThingSpeak. It also provides functions to visualize data stored on ThingSpeak.com or on private installations of the ThingSpeak server.

The ThingSpeak Support Toolbox has these following functions:

  • thingSpeakRead - Read data stored in ThingSpeak channel
  • thingSpeakWrite - Write data to a ThingSpeak channel
  • thingSpeakAuthenticate - Store read/write API key
  • thingSpeakAuthenticatedList - Display list of authenticated channels
  • thingSpeakClearAuthentication - Clear authentication information
  • urlFilter - Scrape numbers from web page
  • thingSpeakPlot - Create 2-D line plot
  • thingSpeakScatter - Create scatter plot
  • thingSpeakStem - Create discrete sequence or stem plot
  • thingSpeakArea - Create filled area 2-D plot
  • thingSpeakPlotYY - Create 2-D line plot with Y axes on both sides


Published with MATLAB® R2016a
