thingSpeakRead

Read data stored in ThingSpeak™ channel

Syntax

[data,timestamps,chInfo] = thingSpeakRead(chId)

[data,timestamps,chInfo] = thingSpeakRead(chId,Name,Value)

Description

[data,timestamps,chInfo] = thingSpeakRead(chId) reads the most recent data from all fields of the specified public channel on ThingSpeak.com and returns the data, timestamp, and channel information.

[data,timestamps,chInfo] = thingSpeakRead(chId,Name,Value) uses additional options specified by one or more Name,Value pair arguments.

Examples

Retrieve the most recent result for all fields of a public channel including the timestamp.

[data,time] = thingSpeakRead(12397);

Retrieve the data for August 8, 2014 through August 12, 2014 for fields 1 and 4 of a public channel, including the timestamp, and channel information.

[data,time,channelInfo] = thingSpeakRead(12397,'Fields',[1 4],'DateRange',[datetime('Aug 8, 2014'),datetime('Aug 12, 2014')]);

Retrieve the last 10 points of data from fields 1 and 4 of a public channel. Return the data and timestamps in a table, including channel information.

[data,chInfo] = thingSpeakRead(12397,'Fields',[1 4],'NumPoints',10,'OutputFormat','table');

Retrieve the last five minutes of data from fields 1 and 4 of a public channel. Return only the data.

data = thingSpeakRead(12397,'Fields',[1,4],'NumMinutes',5,'OutputFormat','table');

Read the last two days of data from fields 1 and 4 of a public channel. Return only the data.

data = thingSpeakRead(12397,'Fields',[1,4],'NumDays',2,'OutputFormat','table');

Retrieve the most recent result for all fields of a private channel.

data = thingSpeakRead(chId,'ReadKey','Your.Read.API.Key','OutputFormat','table');

Retrieve latitude, longitude and altitude data along with the last 10 channel updates for all fields in a public channel and return the data as a table.

data = thingSpeakRead(chId,'NumPoints',10,'Location',true,'OutputFormat','table');

Set the timeout for reading 8000 data points from field 1 of a public channel.

data = thingSpeakRead(12397,'Fields',1,'NumPoints',8000,'Timeout',10,'OutputFormat','table');

Retrieve the last 10 points for all three fields of a public channel on a private ThingSpeak server installation.

myURL = serverURL; % Generate Random Data
[data,time] = thingSpeakRead(6,'NumPoints',10,'URL',myURL);

Input Arguments

chId - The channel identification number specified as a positive integer.

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN .

'DateRange'

Range of data to be returned, specified as a comma-separated pair consisting of 'DateRange' array. The value must have [startdate,enddate], in MATLABÂ® datetime values. ThingSpeak server limits the number of points returned to a maximum of 8000. Adjust your ranges or make multiple calls if you need more than 8000 points of data.

Note: You cannot use DateRange with NumDays or NumMinutes.

Example:thingSpeakRead(12397,'DateRange',[datetime('Aug 8, 2014'),datetime('Aug 12, 2014')]);

'Fields'

Channel field IDs specified as a comma-separated pair consisting of 'Fields' and numeric value. Specify the channel ID from which you need to retrieve data.

Example:thingSpeakWrite(12397,'Fields',[1,3,4]);

'Location'

Positional information of data in the channel, specified as a comma-seperated pair consisting of 'Location' and a logical value. Location information includes latitude, longitude, and altitude.

Example:thingSpeakRead(12397,'Location',true);

'NumDays'

Number of days of data, specified as a comma-separated pair consisting of 'NumDays' and a numeric value, to extract data from. Number of days is measured in 24-hour periods. ThingSpeak server limits the number of points returned to a maximum of 8000. Adjust your number of days or make multiple calls if you need more than 8000 points of data.

Note: You cannot use NumDays with DateRange or NumMinutes.

Example:thingSpeakRead(12397,'NumDays',2);

'NumMinutes'

Number of minutes of data, specified as a comma-separated pair consisting of 'NumMinutes' and a value, to extract data from. Number of minutes of data to retrieve is measured from the current time, (time the command is issued). ThingSpeak server limits the number of points returned to a maximum of 8000. Adjust your number minutes or make multiple calls if you need more than 8000 points of data.

Note: You cannot use NumMinutes with DateRange or NumDays.

Example:thingSpeakRead(12397,'NumMinutes',20);

'NumPoints'

Number of data points, specified as a comma-separated pair consisting of 'NumPoints' numeric value, to extract data from. The number of points or rows of data is counted from the current time (time the command is issued). ThingSpeak server limits the number of points returned to a maximum of 8000. Adjust your number minutes or make multiple calls if you need more than 8000 points of data.

Example:thingSpeakRead(12397,'NumPoints',200);

'OutputFormat'

Class of the output data, specified as a comma-separated pair consisting of 'OutputFormat' and a string value. Valid values are 'matrix' and 'table'. Table is output with [table, channelInfo] and contains the timestamps and the data from the fields of the channel.

Example:thingSpeakRead(12397,'OutputFormat','table');

'Timeout'

Server connection timeout period, specified as a comma-seperated pair consisting of 'Timeout' and a numeric value. The value must the number of seconds for thingSpeakRead to connect to the server.

Example:thingSpeakRead(12397,'Timeout',15);;

'URL'

URL of a private ThingSpeak server, specified as a comma-separated pair consisting of 'URL' and a string value. The value must be a valid URL address.

Example:thingSpeakRead(12397,'URL','http://private.server.url');

'ReadKey'

Read API key of the channel, specified as a comma-separated pair consisting of ''ReadKey' and a string value representing the channel read API key. The Read API key allows you to read data from a private channel. You do not need a Read API key if you are reading data from a public channel. Save your Read API key in a variable for convenience.

Example:thingSpeakRead(12397,'ReadKey','F6CSCVKX42WFZN9Y');

Output Arguments

data - Data from the channel, returned as an array or a table.

timestamps - Timestamp of each data element, returned as an array of datetime.

chInfo - Channel information returned as a structure.
