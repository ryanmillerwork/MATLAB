urlFilter

Scrape numbers from web page

  • Description

  • Output Arguments
  • See Also

Syntax

num = urlfilter(URL,target)

num = urlfilter(URL,target,numNumbers)

num = urlfilter(URL,target,numNumbers,direction)

Description

num = urlfilter(URL,target) returns the first number that appears after the target string.

num = urlfilter(URL,target,numNumbers) additionally specifies the number of numbers to be returned after the target string.

num = urlfilter(URL,target,numNumbers,direction) additionally specifies the read direction, starting from the target string.

Examples

Retrieve the first number after the string 'Latitude' on a specified web page.

url = 'http://www.marinetraffic.com/en/ais/details/ships/219018692';
num = urlfilter(url,'Latitude')

num =

   21.4225

Retrieve the first two numbers after the string 'Latitude / Longitude' on a specified web page.

num = urlfilter(url,'Latitude / Longitude',2)

num =

   21.4225   38.0288

Retrieve the first number before the string 'knots' on a specified web page.

num = urlfilter(url,'knots',1,'backward')

num =

    8.5000

Input Arguments

URL - URL to scrape numbers from, specified as a string.

Example : thingSpeakRead(12397,'URL','http://api.thingspeak.com');

target - Target in the specified URL to scrape data, specified as a string.

numNumbers - Number of targets on the specified URL to read, specified as a numeric value. The argument returns a list of numbers that appear after the target string. The algorithm continues grabbing numbers until the specified numNumbers are grabbed or the call reaches end of the file.

direction - URL read direction, specified as a string value. Valid values are 'forward' or 'backward' . The read command reads in the specified direction, starting from the target string.

Output Arguments

num - Number of targets, returned as a numeric value. The command returns the first number if no optional arguments are specified.
