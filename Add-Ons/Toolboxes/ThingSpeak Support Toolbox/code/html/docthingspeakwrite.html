thingSpeakWrite

Write data to a ThingSpeak™ channel

  • Examples

  • Name-Value Pair Arguments

Syntax

thingSpeakWrite(chId,data)

thingSpeakWrite(chId,data,Name,Value)

response = thingSpeakWrite(_)

Description

thingSpeakWrite(chId,data) writes the specified data to the specified channel.

thingSpeakWrite(chId,data,Name,Value) uses additional options specified by one or more Name,Value pair arguments.

response = thingSpeakWrite(_) returns the response provided by the ThingSpeak server on successful completion of the write operation.

Examples

Write a scalar value to Field 1 of a channel.

thingSpeakWrite(17504,2.3,'WriteKey',23ZLGOCCU9VWHG2H);

Write numeric values to the first 4 consecutive fields [1,2,3,4] of a channel.

thingSpeakWrite(17504,[2.3,1.2,3.2,0.1],'WriteKey',23ZLGOCCU9VWHG2H);

Write non-numeric data to the first 3 consecutive fields [1,2,3] of a channel.

thingSpeakWrite(17504,{2.3,'on','good'},'WriteKey',23ZLGOCCU9VWHG2H);

Write the previous non-numeric values to non-consecutive fields [1,4,6] of a channel.

thingSpeakWrite(17504,'Fields',[1,4,6],'Values',{2.3,'on','good'},'WriteKey',23ZLGOCCU9VWHG2H);

Write latitude and longitude to the channel feed along with values to consecutive fields.

thingSpeakWrite(17504,{2.3,'on','good'},'Location',[-40,23],'WriteKey',23ZLGOCCU9VWHG2H)

Write latitude, longitude, and altitude data to a channel, without adding values to fields.

thingSpeakWrite(17504,'Location',[-40,23,3500],'WriteKey',23ZLGOCCU9VWHG2H)

Write a timestamp for the value being written to a channel. The timestamp provided is interpreted as local time.

tStamp = datetime('2/6/2015 9:27:12','InputFormat','MM/dd/yyyy HH:mm:ss','WriteKey',23ZLGOCCU9VWHG2H);
thingSpeakWrite(17504,[2.3,1.2,3.2,0.1],'TimeStamp',tStamp);

Write values to a channel on a private ThingSpeak installation.

thingSpeakWrite(17504,[2.3,1.2,3.2],'URL','http://myServer.subnet.company.com:3000/','WriteKey',23ZLGOCCU9VWHG2H);

Write a matrix of values to your channel.

data = randi(10,10,3); % Generate Random Data

% Generate timestamps for the data
tStamps = datetime('now')-minutes(9):minutes(1):datetime('now');

channelID = (Enter Channel ID);
writeKey  = (Enter Write API Key);

% Write 10 values to each field of your channel along with timestamps
thingSpeakWrite(channelID,data,'TimeStamp',tStamps,'WriteKey',writeKey);

Write a table of values to your channel.

dataField1 = randi(10,10,1); % Generate random data
dataField2 = randi(10,10,1);

% Generate timestamps for the data
tStamps = [datetime('now')-minutes(9):minutes(1):datetime('now')]';

% Create table
dataTable = table(tStamps,dataField1,dataField2);
channelID = (Enter Channel ID);
writeKey  = (Enter Write API Key);

% Write 10 values to each field of your channel along with timestamps
thingSpeakWrite(channelID,dataTable,'WriteKey',writeKey);

Input Arguments

chId - The channel identification number specified as a positive integer.

data - Data to write to channel, specified as a numeric scalar or numeric vector, a cell array, or a table. If you specify scalar data, the value is written to the first field in the channel (Field1). If you specify a vector or a 1-D cell array, data is written to consecutive fields starting with field 1, for the number of vectors or columns of data.

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN .

'Fields'

Channel fields IDs , specified as a comma-seperated pair consisting of 'Fields' and a 1-by-n positive integer value.

Example: thingSpeakWrite(17504,‘Fields',[1,2,3],'WriteKey','your write api key');

'Value'

Data values, specified as a comma-seperated pair consisting of 'Value' and numeric scalar, numeric vector, a cell array, or table values. Specify the data to be written to channel fields specified by the 'Fields' parameter.

Example: thingSpeakWrite(17504,'Fields',[1,3,4],'Values',[1.1,3.2,4.1],'WriteKey','your write api key');

'Location'

Positional information of data in the channel, specified as a comma-seperated pair consisting of 'Location' and numeric vector. Location information includes latitude, longitude, and altitude.

Example: thingSpeakWrite(17504,(2,3,'on,'good'),'Location',[-40,23,200],'WriteKey','your write api key');

'Timeout'

Server connection timeout period, specified as a comma-seperated pair consisting of 'Timeout' and a numeric value. The value must the number of seconds for thingSpeakWrite to connect to the server.

Example: thingSpeakWrite(17504,(2,3,'on,'good'),'Location',[-40,23,200],'WriteKey','your write api key');

'Timestamp'

Timestamp of data values written to fields in the write channel, specified as a comma-seperated pair consisting of 'Timestamp' and a value. The value must be a MATLAB® datetime value.

Example: thingSpeakWrite(17504,'WriteKey','your write api key','Timestamp',[datetime('2/6/2015 9:27:12')]);

'URL'

URL of a private ThingSpeak server, specified as a comma-separated pair consisting of 'URL' and a string value. The value must be a valid URL address.

Example: thingSpeakWrite(17504,'WriteKey','your write api key','URL','private.server.url');

'WriteKey'

Write API key of the channel, specified as a comma-separated pair consisting of 'WriteKey' and a string representing the channel write key. The API key allows you to write data to a channel. Save your write API key in a variable for convenience.

Example: thingSpeakWrite(17504,'WriteKey','your write api key');

Output Arguments

response - Channel response of the write, returned as a structure. The fields of the structure indicate the values written to the fields of the channel.
