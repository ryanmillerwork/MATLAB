thingSpeakStem

Create discrete sequence or stem plot

Contents

  • Syntax
  • Description
  • Examples
  • Input Arguments
  • Name-Value Pair Arguments
  • See Also

Syntax

thingSpeakStem(Y)

thingSpeakStem(X,Y)

thingSpeakStem(X,Y,Name,Value)

Description

thingSpeakStem(Y) plots the data sequence Y as stems from the x-axis, terminated with circles for the data value. If Y is a matrix, each column is plotted as a separate series.

thingSpeakStem(X,Y) plots the data sequence Y at the values specified in X.

thingSpeakStem(X,Y,Name,Value) produces a stem plot with additional options specified as Name,Value pair arguments.

Examples

Create Stem Plot

Create a step plot of one vector of data.

Y = linspace(-2*pi,2*pi,50);
thingSpeakStem(Y)

Create a Stem Plot with Separate Columns

Create a stem plot of a matrix, plotting each column separately.

X = linspace(0,2*pi,50)';
Y = [cos(X),0.5*sin(X)];
thingSpeakStem(X,Y)

Create a Comparative Stem Plot

Create a stem plot of one set of data values versus another set.

x1 = linspace(0,2*pi,50)';
x2 = linspace(pi,3*pi,50)';
X = [x1,x2];
Y = [cos(x1),0.5*sin(x2)];
thingSpeakStem(X,Y)

Input Arguments

X - X values, specified as a scalar, a vector, or a matrix. X can be a numeric array, logical array, or datetime array.

Y - Y values, specified as a scalar, a vector, or a matrix. Y can be a numeric array, or logical array.

Name-Value Pair Arguments

Specify optional comma-separated pairs of Name,Value arguments. Name is the argument name and Value is the corresponding value. Name must appear inside single quotes (' '). You can specify several name and value pair arguments in any order as Name1,Value1,...,NameN,ValueN.

'Color'

Marker color, specified as one of these values:

  • 'none' - Do not draw the outline.
  • RGB triplet vector or a color string — Specify a custom color. The RGB triplet value of [0 0 0] corresponds to black.

An RGB triplet is a three-element row vector whose elements specify the intensities of the red, green, and blue components of the color. The intensities must be in the range [0,1], for example, [0.4 0.6 0.7]. This table lists RGB triplet values that have equivalent color strings.

'Grid'

Specify if your plot has a grid. Set the value to 'on' to display a grid.

'YLabel'

Y-axis label, specified as a string.

Example: thingSpeakStem(1:10,'YLabel','Y-Axis');

'Xlabel'

X-axis label, specified as a string.

Example: thingSpeakStem(1:10,'XLabel','X-Axis');

'Title'

Title of plot, specified as a string.

Example: thingSpeakStem(1:10,'Title','Plot Title');

'XGrid'

Specify if your plot has a grid only on the X axis. Set the value to 'on' to display an X axis grid.

'YGrid'

Specify if your plot has a grid only on the Y axis. Set the value to 'on' to display a Y axis grid.

'Legend'

Add a legend to your graph, specified as a cell array of strings.

Example: thingSpeakStem(x,y,'Legend',{'Sine','Cosine','Tangent'});
